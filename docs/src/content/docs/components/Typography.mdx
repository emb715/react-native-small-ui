A `Text`-based typography component using `createComponent` variants for preset sizes and font weights. Copy and adapt to your design system.

export const component = `
import { Text as RNText } from 'react-native';
import { createComponent } from 'react-native-small-ui';

const Typography = createComponent(RNText, {
  variants: {
    // Size and line-height scale.
    preset: {
      h1:      { fontSize: 34, lineHeight: 41 },
      h2:      { fontSize: 28, lineHeight: 34 },
      h3:      { fontSize: 22, lineHeight: 28 },
      h4:      { fontSize: 20, lineHeight: 28 },
      h5:      { fontSize: 17, lineHeight: 24 },
      h6:      { fontSize: 14, lineHeight: 18 },
      body:    { fontSize: 16, lineHeight: 24 },
      caption: { fontSize: 12, lineHeight: 18 },
      small:   { fontSize: 10, lineHeight: 14 },
    },
    // Font weight scale. Font family is intentionally omitted —
    // extend() your loaded font family on top of this base.
    weight: {
      thin:      { fontWeight: '200' },
      light:     { fontWeight: '300' },
      normal:    { fontWeight: '400' },
      semibold:  { fontWeight: '600' },
      bold:      { fontWeight: '700' },
      extrabold: { fontWeight: '900' },
    },
  },
  defaultVariants: {
    preset: 'body',
    weight: 'normal',
  },
});
`;

## Usage

All style props, `_light`/`_dark`, `_ios`/`_android`, and `.extend()` work on `Typography` out of the box — it's a full `SmallUIComponent`.

```tsx
// Defaults apply — body size, normal weight
<Typography>Hello world</Typography>

// Explicit preset and weight
<Typography preset="h1" weight="bold">Welcome Back</Typography>
<Typography preset="caption" weight="semibold" _light={{ color: '#666' }} _dark={{ color: '#aaa' }}>
  Last updated today
</Typography>
```

## Adding your font family

`weight` carries only `fontWeight`. To bind your loaded font faces, use `.extend()` at module scope:

```tsx
const Typography = createComponent(RNText, { /* ... variants above ... */ });

// Layer your font family on top — one extend per weight is not needed;
// if your font loads weight automatically via fontWeight, this is all you need:
const AppText = Typography.extend({
  fontFamily: 'Inter',
});

// Or per weight if your font uses separate face files:
const BoldText = Typography.extend({
  fontFamily: 'Inter-Bold',
});
```

## Accessibility — headings

`createComponent` outputs are plain React Native components. To set `accessibilityRole="header"` on heading presets, wrap with a thin component at the call site or in a small helper:

```tsx
import { accessibilityProps } from 'react-native';

const HEADING_PRESETS = new Set(['h1', 'h2', 'h3', 'h4', 'h5', 'h6']);

type TypographyProps = React.ComponentProps<typeof Typography>;

function Text({ preset, ...props }: TypographyProps) {
  return (
    <Typography
      preset={preset}
      accessibilityRole={HEADING_PRESETS.has(preset ?? '') ? 'header' : undefined}
      {...props}
    />
  );
}
```

This keeps `Typography` a pure `createComponent` output — all library features intact — while the thin wrapper handles the one concern that requires render logic.
